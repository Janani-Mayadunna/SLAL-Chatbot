# SriLankan Airlines RAG Chatbot (SLAL-Chatbot)

A Retrieval-Augmented Generation (RAG) chatbot developed as part of the MSc assignment in Natural Language Processing and Generative AI.

The chatbot answers questions related to SriLankan Airlines policies using official airline documentation found in their official website.

## Features

- Retrieval-Augmented Generation (RAG)
- FAISS vector database
- Sentence Transformers embeddings
- FLAN-T5 language model
- Gradio web interface
- Input safeguards and prompt protection
